Structural Tags

  • Header is used to identify what the website is, and what should be on each page to make it look uniform.
  • Nav tag will be the main navigation links that the website will use to organize it's content. Will be on every page of your site. It also makes it easier for search engines to organize and present your information. It only contains links to other pages on it's own website, not usually outside of that.
  • Main will have all the unique information and content that each page needs. Can link to other websites.
  • Footer will have the basic organization information on the bottom, about us, contact, jobs, social media, etc.
  • HTML Validation will check to see if there are any errors in your html code.
